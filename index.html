The W3 schools page I used to walk myself through this

I had a very difficult time trying to figure out what to do this project on. Most UI's that I use don't need much improvement (or at least I think so hence why I use them).

Eventually I decided to do this project on a site called horriblesubs.info; Horriblesubs is a fansub group that translates and subtitles unliscensed TV shows from Japan, Korea, and China into English, so westerners can watch the shows.

The site is not very school appropriate so I wouldn't recommend visiting, but the UI problem is legitimate, and what's more is I am pretty sure I can implement a better solution.

Previously on the site users could navigate to the main directory (figure 1.) click on a show they wanted and a drop down would show with download options for the episode (figure 3).

However, around a year ago they switched it so that when you click on a show in figure 1, you are taken to a second page (figure 2) which has the drop down options for downloads.

For this project I want to recreate the drop down functionality on the index page (figure 1.)
